# GitHub Actions CI/CD with Security Scanning (Ready-to-Use)

## Quick start
1. Create a new public repo on GitHub.
2. Download this zip and extract it.
3. `git init && git add . && git commit -m "feat: initial vulnerable setup"`
4. Add your GitHub remote and `git push -u origin main`.
5. Open the **Actions** tab and watch the run.
6. For Part 3, replace `requirements.txt` with `requirements-secure.txt`, commit, and push.
